# Одностраничное веб-приложение для анализа личных расходов

## Используемые технологии: 
 * Nested BEM
 * HTML/CSS
 * React
 * JSX

## Функционал:
 * Визуальный график личных расходов
 * Добавление нового расхода
 * Динамичная фильтрация данных
